##  Key Features
- **Full AEAT 2026 Compliance**: Generates complete invoices with all mandatory fiscal fields.
- **Privacy First**: 100% Client-side. Your financial data stays in your browser/computer.
- **Verifactu Engine**: Blockchain-like hash chaining and official QR code generation.
- **Portability**: No internet required for core functions. Run it from a folder or USB.
- **Advanced Config**: Integrated IBAN management and simulated Digital Certificate signing.

##  Quick Start
1. **Clone the repo.**
2. **Run the Launcher**:
   -  **Linux**: Double-click `hpro-linux.sh` or use `HaciendaPro.desktop`.
   -  **Windows**: Double-click `hpro-windows.bat`.
3. **Configure & Emit**: Set your data in the Emisor panel and start invoicing!
